## 🚀 Features

- 🔐 **JWT-based Authentication** for Travelers and Admins  
- 🛂 **Role-Based Access Control** (Traveler / Admin)  
- 🧳 **Trip Planning & Booking System**  
- 📅 **View Booked packages**  
- 🧾 **Ratings and reviews**  
- 📱 **Responsive UI** for Mobile & Desktop  

---

## 🛠️ Tech Stack

### 🔷 Frontend:
- React.js  
- React Router  
- Axios  
- Tailwind CSS  

### 🔶 Backend:
- Node.js  
- Express.js  
- MongoDB (Mongoose)  
- JSON Web Token (JWT)  
- Bcrypt.js (Password Hashing)
